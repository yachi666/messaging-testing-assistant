# Messaging Testing Assistant

A React + TypeScript workspace UI for messaging QA operations. It supports three flows:

- User case classification
- User case onboarding automation
- Intelligent test case generation and execution

## Tech Stack

- React
- TypeScript
- Vite

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Notes

LLM-generated outputs should be reviewed by a human before use.
